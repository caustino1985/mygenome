| Attribute | Details |
|-----------|---------|
| **Gene** | COMT (Catechol-O-Methyltransferase) |
| **SNP** | rs4680 (Val158Met polymorphism) |
| **Chromosome** | chr22:19963748-19963749 |
| **Functional Impact** | ↓ Dopamine metabolism in prefrontal cortex |
| **Associated Condition** | ADHD, executive dysfunction |
| **SSA Listing Crosswalk** | 11.00 (Neurological) / 12.02 (ADHD) |
| **Relevant SSA Symptoms** | Disorganization, slowed processing speed |
| **Alleles** | G[A/G]G |

**Legal Significance:**

The COMT Val158Met polymorphism is one of the **most well-established genetic markers for ADHD**. The Val/Val genotype causes rapid dopamine catabolism in the prefrontal cortex (DLPFC), leading to:

- **Executive function collapse** under stress
- **Impaired response inhibition**
- **Difficulty sustaining attention**
- **Disorganization and planning deficits**

**Legal Application:**

This genetic marker **directly supports SSA Listing 12.02 (ADHD)** because it:

1. **Establishes biological basis** for ADHD symptoms
2. **Explains functional limitations** documented in neuroimaging
3. **Proves disability is not subjective** - it's rooted in dopamine metabolism
4. **Defeats "malingering" arguments** - genetic predisposition cannot be faked
5. **Meets SSA standard** for medically determinable impairment

**Expert Witness Testimony:**

> "The COMT Val158Met polymorphism is a well-established genetic marker for ADHD. The Val/Val genotype results in rapid dopamine catabolism in the prefrontal cortex, leading to executive function collapse, particularly under stress. This genetic finding explains the documented functional limitations and directly supports the diagnosis of ADHD under SSA Listing 12.02."
